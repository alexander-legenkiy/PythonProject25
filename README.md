# PythonProject25

## файл masks.py содержит функции, которые принимают на вход номер банковской карты и номер банковского счёта и возвращают их маски

## файл widget.py содержит 1) функицю, которая принимает информацию о счетах и картах и обрабатывает их, возвращает в заданном формате с маской из файла masks.py; 2) функцию, которая принимает на вход дату в определённом формате и возвращает дату в заданном формате

## файл processing.py содержит 1) функцию, которая принимает список словарей и возвращает новый список словарей, исходя из значения ключа state (по умолчанию установлено значение "EXECUTED"; 2) функцию, которая принимает список словарей и возвращает новый список отсортированный по дате, по умолчанию отсортированный по убыванию.

### файлы masks.py, widget.py, processing.py проверены линтерами flake8, mypy, isort, black
