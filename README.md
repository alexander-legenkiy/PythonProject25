# PythonProject25
## Описание:

Данный проект - это функции на языке программирования Python для обработки различной банковской иформации, а именно: 

1. Файл **masks.py** содержит функции, которые принимают на вход номер банковской карты и номер банковского счёта и возвращают их маски

2. Файл **widget.py** содержит 1) функицю, которая принимает информацию о счетах и картах и обрабатывает их, возвращает в заданном формате с маской из файла masks.py; 2) функцию, которая принимает на вход дату в определённом формате и возвращает дату в заданном формате

3. Файл **processing.py** содержит 1) функцию, которая принимает список словарей и возвращает новый список словарей, исходя из значения ключа state (по умолчанию установлено значение "EXECUTED"; 2) функцию, которая принимает список словарей и возвращает новый список отсортированный по дате, по умолчанию отсортированный по убыванию.

## Использование:

+ В файлe **proccessing.py** в переменную _dictionary_list_ скопируйте свой список словарей содержащий инфомацию об id банковской операции, статутсе операции (исполнена - "EXECUTED" или отменена - "CANCELED"), информацию о дате операции. 

+ Функция _filter_by_state(dictionary_list, state)_ возвращает список тех словарей у которых ключ state соответсвует указанному значению: EXECUTED или CANCELED (по умолчанию в функции применяется значение EXECUTED), т.е. функция _filter_by_state(dictionary_list, state)_ отфильтровывает банковские операции по признаку EXECUTED (Исполненные) или по признаку CANCELLED (Отменённые).    

+ Функция _sort_by_date(dictionary_list: list, reverse)_ возвращает список словарей отсортированный по дате банковской операции, в порядке убывания даты, т.е. сначала отображаются последнии операции, а потом операции произведённые ранее. Для того чтобы изменить порядок сортировки от поздних к последним операциям необходимо выбрать значение переменной reverse равное False.
 
+ Файлы masks.py, widget.py, processing.py проверены линтерами flake8, mypy, isort, black
